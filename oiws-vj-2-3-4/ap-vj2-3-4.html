Internet i web

Internet se može definirati na više načina. Internet je javno dostupna globalna paketna podatkovna mreža koja zajedno povezuje računala i računalne mreže ili mreža svih mreža. Internet čini world wide web, koji omogućuje svim korisnicima digitalno dijeljenje informacija. World Wide Web ili www je infrastruktura, odnosno svjetski medij podataka s pomoću kojega korisnici mogu čitati i pisati putem računala koji je spojen na internet.

1. generacija

Prva generacija weba ili web 1.0 se smatra webom statičnog sadržaja (web of content), unutar kojeg se ljudi samo povezuju na web. Stranice su pisane HTML-om, a osnovni komunikacijski protokol je bio HTTP. Stranice su obično definirane sa okvirima (engl. frames), a za pozicioniranje elemenata na stranici koristile su se tablice, dok je navigacija ostvarena direktorijem poveznica.

2. generacija

Druga generacija weba ili web 2.0 se smatra webom komunikacije gdje se ljudi međusobno povezuju. Web više nije statičan, odnosno korisnicima je omogućeno i da oni doprinose stvaranju web sadržaja („read-write web”) čime se briše crta između uobičajenog izdavača sadržaja i korisnika. Web 2.0 započinje 2004.g., a obilježavaju ga razne društvene mreže (Facebook, Twitter, LinkedIn), blogovi, wiki stranice; korisnički doživljaj sučelja stranica je poboljšan, a web postaje otvoren i slobodan. Pojavljuju se razne nove tehnologije:

  • JavaScript
  • Document Object Model (DOM)
  • Ajax
  • Cascading Style Sheets (CSS)
  • Hypertext Markup Language (HTML)
    1. HTML 2 (1995)
    2. HTML 3 (1997)
    3. HTML 4 (1997)
    4. HTML 5 (2014)
  • eXtensible HTML (XHTML)
  • eXtensible Markup Language (XML)
  • eXtensible Stylesheet Language (XSL)
  • Flash
koje su omogućavale prezentaciju i isporuku web usluga bez problema web distribucije.

HTML je opisni jezik za izradu hipertekstualnih dokumenata (izradu web stranica), a dokument HTML-a je tekstualna datoteka (.html). Osnovni elementi HTML-a su oznake HTML-a koje govore web pregledniku što će se prikazati na web stranici. Preglednik obrađuje kôd unutar dokumenta HTML-a i pritom ga prikazuje na pripadni način. HTML pruža statični prikaz sadržaja.

Kaskadni stilovi omogućuju oblikovanje HTML elemenata neovisno o samom sadržaju stranice pa je tako moguće definirati više vrsta stilova za jednu web stranicu. Prije HTML 4 formatiranje stilova je bio dio HTML-a, no potom je prebačeno u CSS zbog jednostavnosti oblikovanja sadržaja, odnosno zbog odvajanja sadržaja i stila. U specifikacijama HTML jezika, još se uvijek navode i atributi za formatiranje načina prikaza sadržaja, ali uz njih se obično navodi da je to zastarjeli način (engl. deprecated). Definicije stilova se nalaze unutar datoteka predlošci stilova (style sheets), a višestruke definicije stilova se potom spajaju u jedan stil (cascading).

3. generacija

Treću generacija weba (ili web konteksta), obilježava povezivanje web aplikacija kako bi se poboljšalo online korisničko iskustvo, a gdje i računala sudjeluju u stvaranju novih web sadržaja. Započinje 2010.g. Oglašavanje se prilagođava ponašanju korisnika, uvodi se inteligentno pretraživanje, preciznije se definira semantika podataka i servisa na webu, omogućuje se dublja analiza podataka odnosno sadržaja, veza i transakcija između ljudi i računala. Dolazi do stvaranja semantičkog weba i personalizacije sadržaja.

4. generacija

Četvrta web generacija se smatra webom stvari, mobilnim webom, koji se prilagođuje svom novom mobilnom okruženju, odnosno povezuje sve uređaje u stvarnom i virtualnom svijetu u stvarnom vremenu (real-time).

5. generacija

Peta generacija weba predstavlja emotivni web, web misli (također se predstavlja i kao Open-Linked and Intelligent Web i kao Symbiotic web, read-write-execution-concurrency web). Web je još u fazi razvoja, a predstavlja povezani web koji komunicira sa nama kao što ljudi međusobno komuniciraju (kao osobni pomoćnik). Fokus weba pete generacije je na emocionalnoj interakciji između ljudi i računala (dnevna aktivnost se temelji na neurotehnologiji). Trenutni web je emotional free, odnosno web koji ne raspoznaje emocije i osjećaje osoba. Korištenjem slušalica korisnici će moći djelovati sa sadržajem koje djeluje sa njihovim emocijama ili promjenama u izrazima lica.



Web mjesto (engl. web site) je jedinstveno ime računala koje ima pridruženu barem jednu IP adresu (primjerice naziv.com, naziv.nesto.com, naziv). Web mjesto se može definirati i kao skup web stranica koji sadrži razne dokumente HTML-a, multimedijalne sadržaje, skripte i ostalo. Web stranice su međusobno povezane poveznicama, čime se ostvaruje hijerarhijska struktura skupa web stranica. Web aplikacija je složeniji skup stranica koji se temelji na specifičnoj programskoj i računalnoj podršci. Web aplikacija omogućuje pristup vlastitim podacima sa bilo koje lokacije koja je spojena na internet te je nije potrebno ručno instalirati, a uspješno radi na bilo kojem uređaju koji ima web preglednik i vezu prema internetu.
